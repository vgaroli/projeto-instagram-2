Crie um MCP que acesse a API pública do Instagram, sem exigir autenticação, para obter todos os vídeos de uma determinada conta. Para cada vídeo dessa lista, traga as informações básicas junto com o número de visualizações e o número de curtidas.

Caso a resposta venha paginada, percorra o máximo de páginas possível. Se ocorrer algum erro de acesso durante esse processo, ignore-o e prossiga, armazenando apenas o que foi possível baixar.

A lista de vídeos deve ser salva em resources/videos/<conta>/lista.csv e organizada por ordem de publicação, do vídeo mais novo para o mais antigo. O MCP poderá ser executado mais de uma vez e, quando isso acontecer, basta atualizar a lista inserindo os vídeos mais recentes no início.

Utilize Node.JS (javascript) e salve o MCP na pasta resources/mcps/<nome do mcp>.
